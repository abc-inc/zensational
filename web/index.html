Here's what makes zensational great:

At Google "10 things" were written down when Google was just a few years old. From time to time this list is revisited to see if it still holds true.
And zensational is developed having this list in mind.

zensational is …

 beautiful
1. Focus on the user and all else will follow.
„Design is all about finding solutions within con­straints. If there were no con­straints, it's not design − it's art.”
—Mathias Duarte, Vice President of Design, Google
 correct
2. It's best to do one thing really, really well.
zensational is built on technology used by Google over a decade. Incremental builds are virtually always cor­rect, it's not necessary to run a clean build "just in case" to ensure cor­rect­ness.
 fast
3. Fast is better than slow.
zensational only rebuilds what is necessary. With optimized de­pend­en­cy a­nal­y­sis, ad­vanced cach­ing and serial ex­e­cu­tion of build ac­tions, you get fast, clean builds.
 modular
4. Democracy on the web works.
A comprehensive, built-in set of mod­ules lets you build web applications for a wide variety of use-cases right out of the box. zensational's extensible mod­ule frame­work, allows you to develop and share your own com­po­nents with other projects.
 lightweight
5. You don't need to be at your desk to need an answer.
zensational is designed for building heavy-duty, mission-critical web applications. But it's great for mobile-friendly websites too. zensational has a 5x lower footprint than for example Material Design.
 reliable
10. Great just isn't good enough.
When you build software with zensational, you're running the same code that has been written and tested by you. Neither black magic nor voodoo is used. It's true!
